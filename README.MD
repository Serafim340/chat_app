# Chat App (Go + WebSocket)

Простой веб-чат на Go с поддержкой комнат и онлайн-пользователей.

## Функционал

* Создание и подключение к чат-комнатам.
* Ввод собственного ника при подключении.
* Список активных пользователей в комнате (автообновление).
* Минимальный фронтенд на HTML/JS + CSS.
* Несколько комнат одновременно.

## Структура проекта

```
chat_app/
├── internal/hub/      # Логика: room, client, hub
├── static/css/        # Стили
├── static/js/         # JS для фронтенда
├── templates/         # HTML-шаблоны
├── main.go            # Точка входа
└── README.md
```

## Запуск

```bash
git clone https://github.com/Serafim340/chat_app.git
cd chat_app
go get github.com/gorilla/websocket
go run main.go
```

Открыть [http://localhost:8000](http://localhost:8000) в браузере.

## Использование

1. Ввести ник и название комнаты или выбрать существующую.
2. Отправлять сообщения через поле ввода.
3. Кнопка в заголовке показывает онлайн пользователей.
